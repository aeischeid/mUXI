About Links

By default links will be loaded via ajax if they point to external content. JavaScipt will do some hide and show work if they point to another section of the page

Add target="_self" to override ajax behavior. Links can also be given target="_blank" just like typical html

Curious what happens when you put a link in a paragraph? see more link type examples. Just be careful of you url path, it gets a little tricky when content is loaded via ajax. You have to remember where you started from.
